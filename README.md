# CypressSample
For POC Comparassion framework tools

## Requirement :
- Node v16++

## Installation :
- Go to folder test
- Run this command in terminal/console:
  ```
  npm install
  ```

## Run App : 
- Go to folder test
- Run this command in terminal/console:
  ```
  npx cypress run
  ```
  
If you want run debuging mode with GUI, run this command :
  ```
  npx cypress open
  ```
